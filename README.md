# Weather App

A responsive weather application. <br>
APIs from: https://openweathermap.org

## Stacks

- React Next.js
- Redux
- TypeScript
- Material UI
- Material UI Chart
- React i18next

## Getting Started

- Use the project node version: `nvm use`
- Install the packages: `npm i`
- Find `.env.template` & set up the .env variables
- Start dev: `npm run dev`
